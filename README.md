## Installation - Pharo 7
Evaluate this code in a Playground to load the Object Miners model and debugger:

```Smalltalk
Metacello new
    baseline: 'ObjectMiners';
    repository: 'github://ClotildeToullec/ObjectMiners';
    load.
```
